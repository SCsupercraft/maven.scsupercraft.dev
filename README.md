# SCsupercraft's Maven Repository

## Groovy DSL

```groovy
repositories {
    maven {
        url "https://maven.scsupercraft.dev/"
    }
}
```

## Kotlin DSL

```kotlin
repositories {
    maven {
        url = uri("https://maven.scsupercraft.dev/")
    }
}
```
